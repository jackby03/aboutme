🛠️ Requisitos Previos

Antes de empezar, asegúrate de tener lo siguiente:

  • 💽 Imagen ISO de Kali Linux: Descárgala desde https://www.kali.org/get-kali/
    → Recomendado: Kali Linux Installer ISO (no la versión live si quieres instalación permanente).

  • 🔧 USB booteable (mínimo 8GB): Herramientas recomendadas para crear el USB:

    • Linux: dd, Etcher o Ventoy

    • Windows: Rufus

    • MacOS: balenaEtcher o terminal con dd

  • 🖥️ Un equipo físico o máquina virtual (VirtualBox o VMware)

🧪 1. Crear USB Booteable

En Linux (el método DEDSEC 💀):
sudo dd if=kali-linux-x.x-amd64.iso of=/dev/sdX bs=4M status=progress && sync

⚠️ Reemplaza /dev/sdX con tu USB (verifícalo con lsblk). Asegúrate de no equivocarte o puedes dañar tus datos.

En Windows:

  1. Abre Rufus

  2. Selecciona tu USB y la ISO de Kali

  3. Esquema de partición: MBR

  4. Sistema de destino: BIOS o UEFI

  5. Clic en Start

🧠 2. Arrancar desde el USB

  1. Inserta el USB y reinicia tu PC

  2. Accede al menú de arranque (generalmente con F12, ESC, F9… depende del equipo)

  3. Selecciona el USB y arranca en modo Kali Linux Installer

⚙️ 3. Instalación paso a paso

Una vez dentro del instalador, sigue estos pasos:

  • Idioma: Español (o el que prefieras)

  • Ubicación: Tu país

  • Teclado: Español - Latinoamericano (o tu distribución)

  • Configuración de red: Elige nombre para el host (ej. dedsec-node)

  • Dominio: Puedes dejarlo en blanco

  • Contraseña de root: Elige una fuerte (dedsec_style!2025)

  • Particionado:

    • Modo fácil: “Guiado - utilizar disco completo”

    • Modo avanzado: Particiona manualmente (/, swap, /home, etc.)

  • Instalar GRUB: Sí, en el disco principal (generalmente /dev/sda)

  • Finaliza y reinicia
